# Enterprise Static CDN with Terraform

## Overview
This project provisions a secure static website hosting infrastructure on AWS using Terraform.

## Services Used
- Amazon S3
- Amazon CloudFront
- Origin Access Control (OAC)
- Terraform
- IAM

## Features
- Private S3 bucket
- CloudFront HTTPS distribution
- Origin Access Control
- Automatic website upload using Terraform
- Server-side encryption
- Bucket versioning
- Public access blocked

## Technologies
- Terraform
- AWS
